# ClawLens Preview

Preview builds of [ClawLens](https://openclaw.ai/) for selected users. Source code is maintained in a separate private repository.

This is the public distribution channel for closed-preview tarballs and (soon) the one-line installer that registers the plugin with your local OpenClaw gateway.

## Status

Initial scaffold only. The installer script and first release tarball are landing in upcoming commits — if you've been pointed here, you're in the right place.
